A beautiful web application for discovering sewing, knitting, and crochet patterns through reverse image search and exploration.

## Features

- **Reverse Image Search**: Upload an image of clothing and find similar patterns
- **Explore Feed**: Browse through curated patterns in a Pinterest-style layout
- **Pattern Details**: View comprehensive pattern information, difficulty levels, and materials
- **Responsive Design**: Beautiful light pink and silver color scheme

## Tech Stack

- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript
- No backend required (template/prototype version)

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/craft-pattern-finder.git
   cd craft-pattern-finder
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. Navigate to `http://localhost:8000`

## Project Structure

- `index.html` - Landing page with search upload
- `explore.html` - Pinterest-style pattern browsing
- `search.html` - Image search results
- `pattern-detail.html` - Individual pattern page
- `css/styles.css` - All styling
- `js/` - JavaScript modules for each page
